# PowerBi
Project Overview
This project leverages the AdventureWorks dataset, a widely used sample database, to design and publish interactive dashboards in Power BI. The goal is to demonstrate data analytics, visualization, and storytelling skills by transforming raw business data into actionable insights.
Key Features
- Data Preparation: Cleaned and structured tables (Sales, Products, Customers, Geography) for analysis.
- Interactive Dashboards: Built multiple reports covering:
- Sales performance by region, product category, and time period
- Customer segmentation and purchasing behavior
- Profitability trends and KPIs
- Visualizations: Used bar charts, line graphs, maps, and slicers for dynamic filtering.
- Business Insights: Highlighted top-performing products, seasonal sales trends, and regional growth opportunities.
Tools & Technologies
- Power BI for dashboard creation and visualization
- SQL for querying and preparing AdventureWorks data
- Excel for initial data exploration and validation
Outcomes
- Demonstrated ability to translate complex datasets into clear, recruiter-friendly dashboards.
- Showcased data storytelling by connecting KPIs with business decisions.
- Built a portfolio-ready project that reflects real-world analytics scenarios.
